The 5-Tool Orchestration Pattern

Instead of building monolithic AI, we created 5 atomic tools:

1. fetch_user_vehicles - Check existing context  
2. extract_vin_with_nova_pro - Capture new data  
3. store_vehicle_record - Build institutional memory  
4. search_web - Validate with real-world data  
5. calculate_labor_estimates - Multi-model consensus  

Key breakthrough: Let Strands agent orchestrate tools dynamically based on conversation context, not rigid workflows.
